# README.md

Этот сайт -- моё личное пространство для текстов, заметок, проектов и нескольких служебных файлов, которые помогают во всём этом ориентироваться. Он устроен как небольшая файловая система: материалы разложены по разделам, у каждой страницы есть метаданные, а длинные тексты могут собираться в дополнительные форматы вроде HTML и PDF.

Интерфейс намеренно сделан в стиле терминала: спокойно, текстово, с акцентом на пути, команды и читаемую структуру вместо декоративных страниц. Версия 1 -- первая законченная версия сайта: секции, локализованные маршруты, сгенерированные страницы, статьи, заметки, теги, скачиваемые файлы и статический вывод уже собраны в одну систему.

## Навигация

**profile** — личный профиль, CV, резюме, публичные ключи и опыт работы;
**articles** — полноценные тексты и статьи;
**notes** — короткие заметки;
**projects** — материалы по проектам;
**tags** — поиск материалов по темам;

## Статус

**version** — 1.0.0;
**content** — секционный, локализованный и собираемый на этапе build;
**output** — статические страницы, готовые к отдаче без клиентского рендера контента;
