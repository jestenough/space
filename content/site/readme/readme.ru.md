# README.md

autophany.space — небольшое пространство для текстов, где LaTeX является основным исходным форматом. Оно ведет себя как спокойная файловая система: служебные файлы живут в `site/`, длинные тексты — в `articles/`, а смысловые связи — в `tags/`.

Проект предпочитает читаемые HTML-страницы для поисковых систем и людей, а PDF-файлы генерируются из тех же LaTeX-исходников как печатные версии.

Гостевые пользователи намеренно имеют доступ только на чтение. Контент можно изучать, открывать, скачивать и индексировать, но нельзя изменять из публичного интерфейса.
