# README.md

Этот сайт -- личное пространство в котором я делюсь с тобой какой-то информацией о себе, какими-то своими знаниями (в виде статей, заметок), и так далее. Он устроен как файловая система: материалы разложены по разделам, у страниц есть структура, теги, метаданные и ссылки на дополнительные форматы вроде PDF.

Интерфейс сделан в стиле терминала: минимум визуального шума, акцент на тексте, структуре и быстром переходе между разделами

## Навигация

**about** — информация о сайте и авторе;
**articles** — полноценные тексты и статьи;
**notes** — короткие заметки;
**projects** — материалы по проектам;
**tags** — поиск материалов по темам;
