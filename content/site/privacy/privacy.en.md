# Privacy Notice

This site is built for reading. It does not use tracking or behavioral analytics.

## What This Site Does Not Do

This site does not set **cookies**.

This site does not load **analytics** scripts.

This site does not embed **tracking pixels**.

This site does not use **advertising identifiers**.

This site does not perform **browser fingerprinting**.

This site does not build **visitor profiles**.

This site does not sell, share, or monetize information about readers.

## What The Browser May Remember

The site may use **localStorage** for interface preferences only:

- selected **theme**;
- preferred **language**;
- desktop panel state;
- reader or zen mode preferences.

These values stay in the browser. They are not sent to the server as a personal profile and are not used to identify a reader.

## Server Requests

The server receives ordinary **HTTP requests** to deliver static files. The site code does not convert those requests into **analytics**, **behavioral tracking**, or **personal profiles**.

## Reader Control

You can remove these local preferences at any time by clearing **site data** or **local storage** for this domain in the browser settings.

If **localStorage** is disabled, the site remains readable. It may only forget interface preferences between page loads.
