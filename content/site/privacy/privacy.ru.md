# Уведомление о приватности

Этот сайт сделан для чтения. Он не использует отслеживание и поведенческую аналитику.

## Чего сайт не делает

Сайт не устанавливает **cookies**.

Сайт не загружает **analytics** scripts.

Сайт не встраивает **tracking pixels**.

Сайт не использует **рекламные идентификаторы**.

Сайт не применяет **browser fingerprinting**.

Сайт не создаёт **профили посетителей**.

Сайт не продаёт, не передаёт и не монетизирует информацию о читателях.

## Что браузер может запомнить

Сайт может использовать **localStorage** только для настроек интерфейса:

- выбранная **тема**;
- предпочитаемый **язык**;
- состояние desktop-панелей;
- настройки reader или zen mode.

Эти значения остаются в браузере. Они не отправляются на сервер как персональный профиль и не используются для идентификации читателя.

## Запросы к серверу

Сервер получает обычные **HTTP-запросы**, чтобы отдать статические файлы. Код сайта не превращает эти запросы в **analytics**, **behavioral tracking** или **персональные профили**.

## Контроль у читателя

Вы можете удалить эти локальные настройки в любой момент через настройки браузера: очистить **site data** или **local storage** для этого домена.

Если **localStorage** отключён, сайт остаётся читаемым. Он может только забывать настройки интерфейса между загрузками страниц.
